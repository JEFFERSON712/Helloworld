
                                 PARA SUBIR UN PROYECTO LOCAL A GITHUB.
1. Lo primero es crear un repositorio, le damos un nombre al repositorio en este caso "HelloWorld".
2. Seleccionamos el proyecto "publico". 
3. Desde la terminal del equipo donde esta el proyecto que queremos subir a github.
4. iniciamos con el comando: git init
5. Lo siguiente es introducir comando: git add . 
6. usamos el comando: git status
7. usamos el comando: git commint -m "Practico"
8. Para conectar el repositorio de git copiamos el link de nuestro repositorio
9. git remote add origin https://github.com/JEFFERSON712/Helloworld
10. Por ultimo realizamos un push: git -u origin master
